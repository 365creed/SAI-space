스케줄 달력 (전체)

정기 프로그램(매주)과 일회 일정이 합쳐져 표시됩니다. 칸을 누르면 해당 날짜가 선택되고 「일회 프로그램 일정」 입력란으로 이동합니다.

회원번호 발급

발급된 번호는 예약 페이지 회원 인증에 사용됩니다.

회원 목록 · 엑셀(CSV)

첫 행: id,name,phone · 업로드 시 동일 id는 덮어쓰기, 없으면 추가됩니다.

    정기 프로그램 (매주 반복)

    교육·건강·체험 분야 정기 프로그램을 텍스트 목록으로 반영했습니다. 첫 방문 시 자동으로 불러오며, 여기서 수정·일괄 등록·주차별 제한을 관리할 수 있습니다.

    요일 (복수 선택)
    주차 제한 (해당 요일이 그 달의 몇 번째인지, 비우면 매주)

    JSON 일괄 반영: 아래에 배열을 붙여넣고 버튼을 누르면 전체가 교체됩니다.
